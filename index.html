There are different programming languages that involve web development.

HTML

HTML is a markup language that acts as the skeleton of both the webpage and the code.
It develops webpages that are bland but are also the bases of all online websites

CSS

CSS is the styler or beauty of the webpages, If HTML is a lady then CSS is the lady with makeup.
It styles and adds attractiveness to the otherwise bland webpage these include:

  • Links
  • Images
  • Back-ground color, font sizes, family fonts etc

JS

This language adds interactivity to the webpage which are normaly just static pages.
It allows developers to create dynamically updating content, use animations, pop-up menus, clickable buttons, control multimedia, etc.
